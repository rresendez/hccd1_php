Main Floodwater Channel Weir 5
Weir System Rehabilitation Project

Entech Engineering Inc.

Updated as of:

2012 Bond List Project No.

April 2017

23b
  • Location
  • Overview
  • Status
  • Photo Gallery

This Weir had severe problems and was declared totally lost during Hurricane Dolly. A new design was proposed to install a brand new weir with steel piling and concrete. This project would also reinforce slopes and flood lines using gabion mattresses and rock rip rap to prevent erosion.

With the new improvements to the ditch bank of adding new rock rip-rap, the length of life of the structure would greatly be improved by preventing scouring around the weir structure.

Project is 100% complete.

Contractor:  Foremost Paving
